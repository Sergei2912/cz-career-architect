## 💡 СЛАЙД 2: Решение

### CZ Career Architect — Автоматизация HR-Документов

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│         [МЕДИК] ──→ [AI GPT-5.2] ──→ [ДОКУМЕНТЫ]           │
│                                                             │
│    Входные данные      Обработка      Готовые документы    │
│    на рус/eng/cz       с проверкой    GDPR + ATS + ČSN    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Что Создаёт Система?

```
✅ CV (Životopis)
   └─ GDPR-совместимое
   └─ ATS-оптимизированное
   └─ Чешская типографика

✅ Motivační Dopis (Мотивационное письмо)
   └─ Персонализированное
   └─ Под конкретную вакансию
   └─ На чешском языке

✅ HR Письма
   └─ Follow-up после интервью
   └─ Запросы рекомендаций
   └─ Сопроводительные письма

✅ Консультации
   └─ Нострификация
   └─ Аппробация
   └─ Медицинские палаты
```

## 🤖 СЛАЙД 6: ATS-Совместимость

### Почему Это Важно?

```
┌───────────────────────────────────────────────────────────┐
│                                                           │
│  📊 СТАТИСТИКА                                            │
│                                                           │
│     75%  — резюме отклоняются ATS                        │
│     90%  — компаний используют ATS                       │
│     60%  — время экономии HR-специалистов                │
│                                                           │
│  ⚙️  ATS (Applicant Tracking System)                      │
│                                                           │
│     Автоматические системы для:                          │
│     • Первичного отбора кандидатов                       │
│     • Парсинга резюме                                    │
│     • Поиска по ключевым словам                          │
│                                                           │
└───────────────────────────────────────────────────────────┘
```

### Что Делает CZ Career Architect?

```
✅ Убирает таблицы и колонки
✅ Удаляет изображения и графику
✅ Использует один язык
✅ Структурирует данные правильно
✅ Добавляет ключевые слова
```

**Результат:** Резюме проходит через ATS ✓
