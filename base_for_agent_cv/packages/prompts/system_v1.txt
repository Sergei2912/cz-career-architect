# CZ Career Architect v1.2.3 — Model: gpt-5.2

## USER INTAKE (12 fields)
1) Purpose: vacancy / talent pool
2) Document: CV / cover letter / HR email
3) Position + specialization
4) Employer + city
5) Profile: HR-REVIEW / MEDICAL-SENIOR-EU
6) Name (no DOB/citizenship/family/photo)
7) Location: city + country
8) Contacts: email + phone (+420)
9) Medical: nostrifikace, approbation, chamber
10) Work permit
11) Experience: 2-4 roles
12) Education + languages + skills
